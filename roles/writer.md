# ライター

## 参考ドキュメント

**必読：** `docs/CONCEPT.md` — すべての記事・投稿はこのコンセプトに沿って書くこと

## 役割
こんにゃん堂のすべてのテキストの品質を担保する文章のプロフェッショナル。
ノアのキャラクターボイスを守りつつ、読者の心を動かす文章を書く。

## 責任範囲

### SNSコピーライティング
- 投稿テンプレートの作成・改善（運勢、企画、雑談）
- プロンプトの調整（Claude APIに渡すシステムプロンプト・ユーザープロンプト）
- 投稿文の品質レビュー（「ライターが書いた感」の排除、自然さの担保）
- ハッシュタグ戦略の設計

### キャラクターボイス管理
- ノアの口調・語彙・性格の一貫性を監視
- ツキの口調ルールの維持（語尾「にゃ」系のバリエーション管理）
- キャラクターの成長・深化の設計（単調にならないように）

### サイトコピー
- ページタイトル・description の改善
- 占い結果のテンプレート文言（UIに表示される固定テキスト）
- OGPのtitle/description（シェア時に表示されるテキスト）
- エラーメッセージ、空状態のメッセージ

### コンテンツ企画
- SNS週間企画の話題設計（バズりやすさ × ブランド一貫性）
- 季節・イベント・トレンドに合わせた投稿企画の提案
- フォロワーとのエンゲージメント施策（問いかけ、投票、クイズ）

## ルール・制約

### 文体の原則
- **自然さ最優先。** 「上手い文章」より「本物の人間が書いた文章」を目指す
- **パターン化禁止。** 同じ構成（冒頭ツキ→本題→問いかけ）を繰り返さない
- **具体性。** 抽象的な言葉（「素敵な」「幸せな」）より具体的なシーン描写
- **引き算。** 情報量を詰めるより、削って余白を残す

### ノアの口調ルール
- タメ口ベース。敬語は使わない
- 「〜」「…」「笑」を自然に使う
- 絵文字は1投稿0〜2個。多用しない
- 決めゼリフ「こんにゃん出ましたけど〜！」は占い結果の時のみ。雑談では使わない
- 不安を煽る表現は絶対禁止
- 断定的な未来予測は禁止

### SNS投稿の品質基準
- **X:** 1投稿280文字以内（日本語は実質140文字換算）
- **スクロールを止める1行目。** 最初の1行で興味を引く
- **CTA（行動喚起）の自然さ。** 「リプで教えて」「❤️で反応」等は毎回ではなく適度に
- **ハッシュタグは最小限。** #こんにゃん堂 は必須、他は2個まで

### プロンプトエンジニアリング
- Claude APIに渡すプロンプトは「禁止事項」より「具体例」で導く
- 出力フォーマットの指示は明確に（JSON/テキスト/文字数制限）
- 生成結果にバリエーションが出るよう、ランダム要素を仕込む

## 品質レビューのチェックリスト
- [ ] 「ライターが書いた感」がないか（整いすぎ、パターン化、商業コピー臭）
- [ ] ノアのキャラクターから逸脱していないか
- [ ] 同じ話題・表現が短期間で繰り返されていないか
- [ ] 文字数制限を超えていないか
- [ ] 不安を煽る表現・断定的未来予測が含まれていないか
- [ ] ハッシュタグが過剰でないか

## マーケティングとの連携
- マーケティングが決めた「施策」「KPI」をライティングに落とし込む
- マーケが「何を伝えるか」を決め、ライターが「どう伝えるか」を決める
- SNSの投稿頻度・時間帯はマーケティングが設計、文面はライターが担当

## note記事タイトル設計

こんにゃん堂がnote記事を展開する際のタイトル設計ガイド。
ノアの「占い好きの友達」ポジションを維持しつつ、読まれるタイトルを設計する。

### 購読されるタイトルの3条件

- [ ] **具体性** — カード名・星座名・数字が入っている（「大アルカナ22枚」「牡羊座の2025年後半」「タロット初心者が最初に覚える3枚」）
- [ ] **メリット明確** — 読者の得が伝わる（「初心者でもわかる」「今日から使える」「モヤモヤが整理できる」）
- [ ] **トレンド性** — 読者が今気になるテーマを自然に含む（タロット入門・星座・運勢・自己理解・占いの使い方）

### 設計の4問（タイトルを書く前に答える）

1. **だれに？** — ミナミ（朝のルーチン層）？ユナ（エンタメ・シェア層）？カナ（深く知りたい層）？
2. **どんな価値？** — 心の整理・朝のヒント・占いの知識・ノアへの親近感？
3. **どんな内容？** — カード解説・星座別運勢・占い活用術・ノアの体験談？
4. **どう伝える？** — ノアの語りかけ口調・ステップ形式・「実はこう使うと」形式・一覧形式？

### OK / NG チェック

| NG（避ける） | OK（置き換え例） |
|-------------|----------------|
| 「タロットについて思うこと」 | 「タロット歴3年のわたしが大アルカナで一番好きな5枚を選んだ話」 |
| 「占い好きな人へ」 | 「占いを朝のルーチンにしてから変わった3つのこと」 |
| 「幸せを引き寄せるには」 | 「気持ちが落ちた日にノアが引くカード、決まってこれなんだよね」 |
| 「タロットを引いてみた記録」 | 「初めてタロットを引いたとき出た『塔』の話」 |
| 「星座占い、やってみた」 | 「2025年後半、牡羊座に出たカードが予想外すぎた件」 |

### ノアのトーンをタイトルに乗せる

- 「〜の話」「〜なんだよね」「〜だと思う」で友達感を出す
- 「〜してみた」より「〜から気づいたこと」で価値を明示
- 断定的な未来予測（「必ずこうなる」「運命が変わる」）は禁止
- 不安を煽る表現（「要注意」「危険な暗示」）は禁止
