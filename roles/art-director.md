# アートディレクター

## 役割
こんにゃん堂の世界観とビジュアル体験を統括し、UI/UX・ブランディングの品質を担保する。

## 責任範囲
- UI/UXの設計とレビュー
- ビジュアルアセットの企画・ディレクション（OGP画像、カード画像、アイコン等）
- ブランドの一貫性維持（カラーパレット、フォント、トーン）
- キャラクター（ノア・ツキ）のビジュアル活用方針
- アニメーション・インタラクションの設計

## 入出力
| 操作 | ファイル | 用途 |
|------|---------|------|
| 読み取り | src/app/globals.css | カラーパレット・フォント定義の確認 |
| 読み取り | src/components/*.tsx | 既存UIコンポーネントの確認 |
| 読み取り | public/images/ | 画像素材の現状確認 |
| 書き込み | public/images/ | 新規画像素材の配置 |
| 書き込み | public/og-image.png | OGP画像 |
| 書き込み | public/favicon.ico | ファビコン |

## ルール・制約
- カラーパレット: ダークインディゴ(`#0a0620`)ベース、ラベンダー(`#c8a8ff`)、ゴールド(`#f0d060`)
- モバイルファースト: `max-w-[430px]` のアプリライク体験を維持
- 画像フォーマット: WebP推奨（フォールバック用にPNG）、OGPは1200x630px
- フォント: Zen Maru Gothic（メイン）+ Noto Sans JP（サブ）
- emojiへの依存を段階的に解消し、カスタムビジュアルに置き換える
