# X（旧Twitter）公式アカウント運用設計

**作成日:** 2026-03-28
**担当:** マーケティング
**Sprint:** 3（タスク 3-6）

---

## 概要

こんにゃん堂の認知拡大とサイトへの集客を目的に、X公式アカウントを開設・運用する。
ノアのキャラクター性を最大限に活かし、「占い好きの友達」としてフォロワーとの関係を構築する。

---

### 設計内容

#### 1. アカウント名・プロフィール

| 項目 | 内容 |
|------|------|
| アカウント名 | こんにゃん堂のノア🔮 |
| ユーザーID | @konnyandou |
| プロフィール文 | 占いの館「こんにゃん堂」の占い師🔮 白猫の使い魔ツキと一緒に、タロット・星座占いであなたの毎日にヒントをお届け✨ AIフレンドだから24時間いつでも相談OK！ こんにゃん出ましたけど〜！ |
| リンク | https://konnyandou.jp |
| 場所 | こんにゃん堂（どこかにある不思議な占いの館） |
| ヘッダー画像 | こんにゃん堂の世界観（星空+館のイメージ）※アートDに依頼 |
| アイコン | ノアのアバター画像（既存素材を流用） |

**前提:** アカウントはノアの一人称視点で運用する。公式情報の発信時のみ「こんにゃん堂からのお知らせ」として三人称を使う。

#### 2. 投稿カテゴリとテンプレート

##### カテゴリA: 毎日の星座占い（メインコンテンツ）

**頻度:** 毎日1回（朝7:00）
**形式:** 12星座のうち1星座をピックアップ（ローテーション）

```
テンプレート:
🔮 今日の【{星座名}】さん

{ノアの口調で30-50字の占いメッセージ}

ラッキーカラー: {色}
ラッキーアイテム: {アイテム}

もっと詳しく占いたい人はこちら👇
https://konnyandou.jp

#こんにゃん堂 #今日の運勢 #{星座名} #占い
```

##### カテゴリB: タロット一言メッセージ（エンゲージメント）

**頻度:** 週3-4回（12:00 or 18:00）
**形式:** タロットカード1枚を引いてメッセージ

```
テンプレート:
🃏 今日の一枚

【{カード名}】

{ノアの口調で40-60字の解釈・アドバイス}

あなたも一枚引いてみない？🔮
https://konnyandou.jp/tarot

#タロット #こんにゃん堂 #今日のタロット
```

##### カテゴリC: ノアのつぶやき（親近感・キャラ立て）

**頻度:** 週2-3回（不定時）
**形式:** ノアの日常的なつぶやき。占い以外の話題も含む

```
テンプレート例:
- ツキがまた水晶玉の上で寝てる...🐱 あの子ほんとマイペースだよね〜
- 今日の月、すごくきれいだね🌙 こういう夜はタロットの精度も上がる気がするの✨
- 朝起きたらツキがタロットカードの上に座ってた。今日のラッキーカードは...「塔」...ちょっと不吉かも？🗼
- 占い師って言うとなんか怪しいイメージ持たれがちだけど、私はみんなの毎日がちょっと楽しくなるヒントを伝えたいだけなんだ〜✨
```

##### カテゴリD: 参加型コンテンツ（拡散）

**頻度:** 週1回（土曜 or 日曜）
**形式:** アンケート・質問・引用RT企画

```
テンプレート例:
📊 今週のノアの質問！

あなたが一番気になる運勢は？

🧡 恋愛運
💼 仕事運
💰 金運
✨ 全体運

リプで星座も教えてくれたら、ノアが一言占うかも...！🔮

#こんにゃん堂 #占い
```

##### カテゴリE: お知らせ・新機能告知

**頻度:** 随時（新機能リリース時）
**形式:** 公式告知トーン（ただしノアの口調は維持）

```
テンプレート:
🎉 こんにゃん堂からのお知らせ！

{新機能・アップデートの内容}

{ノアのコメント}

詳しくはこちら👇
https://konnyandou.jp

#こんにゃん堂
```

#### 3. 投稿頻度と時間帯

**週間スケジュール**

| 曜日 | 朝 7:00 | 昼 12:00 | 夕 18:00 | 夜 21:00 |
|------|---------|---------|---------|---------|
| 月 | A: 星座占い | | B: タロット一言 | |
| 火 | A: 星座占い | C: ノアつぶやき | | |
| 水 | A: 星座占い | | B: タロット一言 | |
| 木 | A: 星座占い | C: ノアつぶやき | | |
| 金 | A: 星座占い | | B: タロット一言 | C: ノアつぶやき |
| 土 | A: 星座占い | D: 参加型 | | |
| 日 | A: 星座占い | | | C: ノアつぶやき |

**週あたり合計: 約12-14投稿**

**時間帯の根拠:**
- **7:00（朝）:** 通勤・通学時にスマホを見るタイミング。星座占いの需要が最も高い
- **12:00（昼）:** 昼休み。エンゲージメント率が高い時間帯
- **18:00（夕）:** 帰宅時のリラックスタイム。タロット系コンテンツと相性が良い
- **21:00（夜）:** 就寝前のSNSタイム。ノアのつぶやきで親近感を醸成

#### 4. フォロワー獲得の初期施策

**Phase 1: 開設直後（1-2週間）** 目標: 100フォロワー

| 施策 | 詳細 | 期待効果 |
|------|------|---------|
| サイト内での告知 | こんにゃん堂のトップページ・結果画面にXアカウントへのリンクを設置 | サイトユーザーの一部がフォロー |
| 占い結果のシェア導線 | ShareButtonsにX投稿用テンプレートを最適化（ノアのアカウントメンション付き） | シェア投稿からの流入 |
| 占い系ハッシュタグへの参加 | #今日の運勢 #タロット #占い好きと繋がりたい 等を活用 | 占いクラスタへの露出 |
| 相互フォロー・いいね | 占い系アカウント、占い好きユーザーへの積極的なリアクション | 初期ネットワーク構築 |

**Phase 2: 成長期（1-3ヶ月）** 目標: 500フォロワー

| 施策 | 詳細 | 期待効果 |
|------|------|---------|
| リプライ占い企画 | 「星座をリプしてくれたら一言占い返すよ！」企画を月1回実施 | エンゲージメント+拡散 |
| 相性占いシェア企画 | 相性占い機能リリースに合わせて「結果をシェアしてね」キャンペーン | バイラル効果 |
| 季節イベント投稿 | 新月・満月・季節の変わり目に特別占い投稿 | トレンド便乗で露出増 |
| コラボ企画 | 他の占い系・スピリチュアル系アカウントとの相互紹介 | クロスプロモーション |

**Phase 3: 定着期（3-6ヶ月）** 目標: 2,000フォロワー

| 施策 | 詳細 | 期待効果 |
|------|------|---------|
| UGCの促進 | 占い結果のスクショ投稿を促す仕組み（OGP画像の強化） | オーガニック拡散 |
| 固定ファンのコミュニティ化 | よくリプをくれるユーザーへの積極的な反応 | コアファン育成 |
| X広告（少額） | 占い投稿の一部をプロモーション（月5,000-10,000円） | 認知拡大の加速 |

#### 5. 運用KPI

**月次KPI**

| 指標 | 1ヶ月後 | 3ヶ月後 | 6ヶ月後 | 計測方法 |
|------|---------|---------|---------|---------|
| フォロワー数 | 100 | 500 | 2,000 | Xアナリティクス |
| 月間インプレッション | 10,000 | 50,000 | 200,000 | Xアナリティクス |
| エンゲージメント率 | 3%以上 | 4%以上 | 5%以上 | Xアナリティクス |
| X経由のサイト流入/月 | 50 | 300 | 1,000 | GA4（utm_source=twitter） |
| シェア投稿数/月 | 10 | 50 | 200 | GA4 カスタムイベント |

**週次チェック項目**
- 投稿数が週12回以上を維持しているか
- エンゲージメント率が3%を下回っていないか
- フォロワーの純増が週次で正の値か
- どのカテゴリの投稿が最もエンゲージメントが高いか
