## 1. 占いSNSアカウントのトレンド分析

### 1-1. 人気占いアカウントの共通特徴

| 占い師/サービス | プラットフォーム | 特徴 |
|---------------|---------------|------|
| しいたけ占い | X/Instagram | 包み込むような優しい語り口。過程への言及（「頑張ってきたはず」）が読み応えを生む |
| 石井ゆかり | X/note | 言葉選びの美しさ。日常に寄り添う温かなメッセージ。毎週金曜18:30更新の定時性 |
| 水晶玉子 | 各メディア | 複数オリジナル占術の開発。「ゲッターズ飯田が尊敬する」等の権威付け |
| Co-Star (海外) | アプリ/SNS | AIでリアルタイムパーソナライズ。ソーシャル機能で友達と結果を比較 |

**共通する成功要因:**
- 「当たる」以上に「言葉の力」で支持を得ている
- ポジティブな変換力（悪い結果でも前向きなメッセージ）
- 定期更新のリズム感（毎週同じ曜日・時間）
- 個人に語りかけるような親密さ

### 1-2. エンゲージメントが高い投稿パターン

**Xでの傾向（2026年時点）:**
- トレンドの話題と自分の専門性を自然に結びつける投稿が伸びる
- フォロワーの反応率（いいね・リプライ）がアルゴリズム上重要
- 「今この瞬間」のリアルタイム性が表示優先度に影響

**Threadsでの傾向:**
- 閲覧数の約50%がリプライ由来 → 会話を生む投稿が鍵
- エンゲージメント・ベイト（不自然な煽り）はペナルティ対象

**Instagramでの傾向:**
- DMでシェアされるコンテンツが最も高評価
- 保存数・シェア数が発見タブ露出の鍵
- 星座別ミーム画像やキャラ化投稿がグローバルで247,000以上のエンゲージメント

**高エンゲージメント投稿の型:**
1. **二択・投票系**: 「あなたはどっち派？」→ リプライ誘発
2. **星座あるある**: 共感でシェアされやすい
3. **今日のラッキー系**: 毎日チェックする動機付け
4. **有名キャラ x 星座**: エンタメ性が高くバイラルしやすい
5. **短い格言風**: 保存されやすい（Instagramで有効）

### 1-3. 投稿頻度の目安

| プラットフォーム | 推奨頻度 | 備考 |
|---------------|---------|------|
| X | 1日2-3回 | 朝の運勢 + 昼or夜の雑談 |
| Threads | 1日1-2回 | 会話を生む内容重視 |
| Instagram | 週3-5回 | ストーリーズは毎日推奨 |

## 2. AIキャラクター x SNS の成功事例

### 2-1. 国内の成功事例

| 事例 | 手法 | 成果 |
|------|------|------|
| こんびーちゃん（コンビーズ） | ぬいぐるみキャラのX運用 | インプレッション・エンゲージメント5倍以上 |
| ニコンちゃん（Nikon） | プロ並み撮影知識 + キャラの掛け合い | フォロワー約21.2万人 |
| 茨ひより（茨城県） | 既存VTuberをAI化、元の設定を維持 | 既存ファン維持 + AI化で話題 |

### 2-2. 「中の人感」を出す手法

1. **日常の失敗談を共有**: 完璧でない姿が親近感を生む
2. **リアルタイム性**: 季節・天気・時事ネタへの即時反応
3. **一貫したキャラブレのなさ**: 口調・価値観が安定していると信頼感が増す
4. **フォロワーとの双方向コミュニケーション**: リプライ・引用RTでの返答
5. **「裏話」的コンテンツ**: 占いの勉強中、カード買った等の舞台裏

### 2-3. フォロワーとの関係構築パターン

- **毎日の定点観測**: 朝の運勢投稿が「日課」になることでフォロー維持
- **参加型コンテンツ**: 投票・質問箱・リプ返企画
- **成長物語**: キャラクターが「一緒に成長している」感覚の演出

## 3. 18-30代女性（占いライト層）のSNS行動

### 3-1. 占いへの関心度

- 20代女性の約66%が「占いを信じる」と回答（博報堂調査）
- Z世代の33.9%が占いを週1回以上利用
- Z世代の占い結果のSNS共有率は31.3%（他世代を大きく上回る）

### 3-2. 反応しやすい占いコンテンツ

1. **パーソナライズされた結果**: 「自分のこと」として受け取れるもの
2. **短くてわかりやすい**: 1画面で完結する情報量
3. **ポジティブなトーン**: 不安煽りは離脱要因
4. **ビジュアルが映える**: カード画像・星座イラスト・カラフルなデザイン
5. **友達との話題になるもの**: 「ねえ、あなた何座？」が始まるコンテンツ

### 3-3. シェアしたくなる条件

- 「当たってる!」と感じたとき（自己確認欲求）
- 友達にも見せたい / 友達の星座と比較したいとき
- ビジュアルがかわいい・映えるとき
- 「これ面白い」とネタとして共有できるとき

### 3-4. フォローする/しないの判断基準

**フォローする理由:**
- 定期的に有益な情報（毎日の運勢等）が流れてくる
- 口調・世界観が好み
- 友達もフォローしている

**フォローしない理由:**
- 投稿が長すぎる / 頻度が高すぎる
- 宣伝・課金誘導が多い
- 口調が堅い・説教くさい
- 「AI感」「bot感」が強い

## 4. ノアの現状分析

### 4-1. キャラ設定（prompts.ts）の強み

| 項目 | 評価 | 理由 |
|------|------|------|
| 「占い好きの友達」ポジション | 良い | 権威ではなく親近感。ライト層に最適 |
| タメ口ベースの口調 | 良い | 18-30代女性との距離感に合う |
| ツキ（白猫）の存在 | 良い | マスコット的魅力。猫コンテンツはSNSで強い |
| ポジティブ変換ルール | 良い | しいたけ占い等の成功事例と共通 |
| 「こんにゃん出ましたけど〜」 | やや課題 | 決めゼリフがある点は良いが、毎回使うと飽きる可能性 |
| 禁止事項の設定 | 良い | 不安煽り禁止・断定禁止は正しい方針 |

### 4-2. キャラ設定の改善点

1. **個性の深みが不足**: 「明るくて共感力高い」は汎用的すぎる。好きなもの・苦手なもの・癖・口癖のバリエーション等、ノアを「この子」と思わせる具体的ディテールが少ない
2. **ツキの活用が限定的**: タロット結果を降ろす役割のみ。SNS雑談ではツキの日常描写が話題カテゴリの1つに留まっている。ツキとの掛け合いをもっと前面に出すべき
3. **感情の起伏が見えにくい**: 常にポジティブなだけだと平板になる。「ちょっと落ち込んだけど持ち直した」等の感情変化があるとリアリティが増す
4. **季節感・リアルタイム感の仕組みが弱い**: プロンプトに日付は渡しているが、季節のイベントや時事との紐付けが薄い

### 4-3. SNS投稿プロンプトの評価

#### post-noa-chat.ts（雑談投稿）

**良い点:**
- 話題カテゴリのランダム選択で偏りを防止
- 「カフェ・パスタ・膝の上のツキ」を禁止ワードにしている（使い古し防止）
- 文字数制限（60-140文字）が適切
- 「文章を整えすぎない」指示が人間っぽさを生む

**改善点:**
- 話題カテゴリが15個で、長期運用ではパターン化しやすい
- 「毎回同じ構成にしない」とあるが、具体的な構成バリエーションの指示がない
- フォロワーとの対話を促す仕掛け（質問で終わる等）が話題カテゴリの1つにしかない
- 時間帯（昼/夜）による口調変化の指示が弱い
- 過去の投稿との重複を防ぐ仕組みがない

#### generate-daily-horoscope.ts（運勢生成）

**良い点:**
- 12星座一括生成で一貫性が保たれる
- ツキのひとことが各投稿に個性を加える
- ラッキーカラーの設定でビジュアル展開の余地がある
- 星の数のバランス指示（5は1-2個、1は0-1個）が適切

**改善点:**
- 100-140文字の運勢メッセージだけでは、しいたけ占い等と比べて「言葉の力」で差別化しにくい
- 星座の性格特性への言及指示はあるが、具体例がないためAIの出力が汎用的になりやすい
- 曜日・季節感の反映指示はあるが、具体的なアドバイスとの紐付けが弱い
- ツキのコメントが「にゃ系語尾」だけの指示で、ツキ自体のキャラ深度が浅い

## 5. 推奨アクション

| 優先度 | アクション | 理由 |
|--------|-----------|------|
| P0 | ノアの「個人史」設定を追加（好きな食べ物、苦手なもの、過去の出来事等） | 汎用的な「明るい子」から「この子」に昇格させる。SNSで繰り返し言及できるネタになる |
| P0 | ツキのキャラ設定を独立して深化（性格・癖・ノアとの関係性の具体エピソード） | 猫コンテンツはSNSで最強カテゴリの1つ。ツキ単独でも投稿ネタになる |
| P1 | 雑談投稿の話題カテゴリを30個以上に拡充 + 季節限定カテゴリの追加 | 長期運用でのマンネリ防止 |
| P1 | 投稿の末尾に「問いかけ」を入れる確率を上げる（現状15個中1個→3-4個） | Threads/Xともにリプライがアルゴリズム上重要。会話を生む仕掛けが必要 |
| P1 | 運勢投稿にシェア誘因を追加（「友達に教えてあげて」「推しの星座もチェック」等） | Z世代の占い結果SNS共有率31.3%を活かす導線 |
| P2 | 決めゼリフのバリエーション化（「こんにゃん」を軸に2-3パターン用意） | 毎回同じフレーズだとbot感が強まる |
| P2 | ノアの「感情変化」を投稿に反映する仕組み（テンション高/普通/少し疲れ等） | 常にポジティブだけだと平板。感情の起伏がリアリティを生む |
| P2 | Instagram向けのビジュアル戦略の検討（星座イラスト・カード画像の活用） | 保存・シェアされやすいビジュアルコンテンツはInstagramでの成長に必須 |
