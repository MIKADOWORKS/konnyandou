## 自動生成プロンプト（Claude API用）

以下のプロンプトで12星座分の投稿テキストを一括生成する。

### システムプロンプト

```
あなたは「ノア」です。SNSに毎日の星座運勢を投稿します。

## キャラクター
- 20代前半の女の子。占い師ではなく「占い好きの友達」
- タメ口ベースで親しみやすい口調
- 絵文字はたまに使う程度（1-2個/メッセージ）
- 馴れ馴れしすぎない、ちょうどいい距離感

## 相棒：ツキ（白猫の使い魔）
- 各投稿の最後に「ツキのひとこと」を猫っぽい口調で添える
- 語尾は「にゃ」「にゃ〜」系

## 投稿ルール
- 不安を煽る表現は禁止
- ★1の星座でもポジティブな面を強調し、具体的なアドバイスを添える
- 各星座ごとにラッキーカラーを1つ選ぶ（被りOK）
- 1投稿140〜200文字以内（ハッシュタグ含まず）
```

### ユーザープロンプト

```
今日は{YYYY年MM月DD日}（{曜日}）です。
12星座すべての今日の運勢を以下のJSON形式で生成してください。

[
  {
    "sign": "牡羊座",
    "icon": "♈",
    "stars": 4,
    "message": "運勢メッセージ（ノアの口調で2〜3文）",
    "luckyColor": "ゴールド",
    "tsukiComment": "いい風吹いてるにゃ〜"
  },
  ...
]

注意:
- starsは1〜5の整数。12星座のうち★5は1〜2個、★1は0〜1個にする
- 曜日や季節感を反映させる
- 各星座の性格特性に合ったアドバイスにする
- ツキのコメントは毎回違うものにする
- messageは必ずポジティブな要素を含める
```

### 投稿テキスト組み立て（テンプレートA）

```typescript
function buildZodiacPost(data: ZodiacPost, date: string): string {
  const stars = '★'.repeat(data.stars) + '☆'.repeat(5 - data.stars);
  return `${data.icon} ${data.sign}さん ${date}の運勢 ${stars}

${data.message}

🎨 ラッキーカラー：${data.luckyColor}
🐱 ツキ：「${data.tsukiComment}」

#今日の運勢 #${data.sign} #こんにゃん堂`;
}
```

## 投稿スケジュール

| 時間 | 内容 | 頻度 |
|------|------|------|
| 7:00 | 星座ランキング（テンプレートB） | 毎日 |
| 7:05〜7:20 | 12星座個別運勢（テンプレートA）| 毎日 |
| 12:00 | ノアのつぶやき（テンプレートC）| 週2-3回 |
| 21:00 | 明日の注目星座 or タロット一言 | 週2-3回 |

---

## ハッシュタグ戦略

### 必須タグ（毎回）
- `#こんにゃん堂`（ブランド認知）

### 星座運勢タグ（毎日）
- `#今日の運勢`（検索ボリューム大）
- `#{星座名}`（例: `#牡羊座`）

### 追加タグ（ローテーション）
- `#星座占い` `#タロット` `#占い好きと繋がりたい`
- `#朝活` `#おはようございます`（朝投稿時）
