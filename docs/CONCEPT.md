## 1. 誰に（Target Audience）

**メインターゲット：** 18〜26歳女性（Z世代・SNSネイティブ）

**サブターゲット：** 27〜35歳女性（ミレニアル世代・仕事・ライフステージの転換期）

**ターゲット像（3ペルソナ）：**

| ペルソナ | 年齢 | 一言 | 占いの使い方 |
|---------|------|------|------------|
| ミナミ（コア層） | 24歳・メーカー勤務 | 朝の通勤で習慣として占いチェック | 「今日のヒント」として気軽に楽しむ |
| ユナ（拡散層） | 19歳・大学生 | 友達とのLINEで占い結果をシェア | ネタになるかどうかが基準 |
| カナ（リテンション層） | 28歳・デザイナー | 仕事ストレスの夜、心の整理に | タロットの深みを知りたい派 |

**共通の行動パターン：** X・Instagramでのながら消費 / 手軽にサッと使いたい / シェアのネタを探している

## 2. どんな痛みを解決するか（Pain）

**具体的な痛み：**

1. **迷いが多い、でも相談できない** — 進路・恋愛・仕事の小さな判断に確信が持てない。友達や家族には大げさで言えない
2. **お金を払うほどではない** — 本格鑑定は高い・怖い・面倒。「ちょっと背中を押してほしいだけ」
3. **占いサービスが重い・冷たい** — メディア系占いはコンテンツ量が多く探しにくい。AI占いはキャラがなく素っ気ない
4. **朝のルーチンに使えるものがない** — 星座占い記事はSEO量産感があり、読む気がしない
5. **誰かと感情を共有したい** — 結果を共有する相手やリアクションがほしい。孤独感のある悩み相談

**痛みの根本原因：**
既存の占いサービスは「権威的な占い師」か「無機質なAI」のどちらか。
「友達感覚で気軽に話せる、でもちゃんと占いを知ってる人」が存在しない。

## 3. なぜ こんにゃん堂 だけが解決できるか（USP）

**こんにゃん堂の固有の強み：**

- **「占い好きの友達」ポジション** — 占い師でも権威でもない。タメ口で、一緒に考えてくれる存在
- **ノア + ツキのキャラクター** — 23歳女性ノアと白猫ツキという具体的な世界観。感情移入しやすく、投稿がコンテンツになる
- **24時間・インストール不要** — URLを開くだけ。深夜の悩みにも即対応
- **知識の裏付けがある** — タロット図像学・カバラを知りながら、普段はひけらかさない。ペルソナCにも耐える深さ

**競合との比較：**

| | こんにゃん堂 | LINE占い / cocoloni | Yahoo!占い | ChatGPT直接 |
|---|---|---|---|---|
| 親近感 | ◎（友達） | △（サービス） | ✕（メディア） | ✕（ツール） |
| キャラクター | ◎ | △ | ✕ | ✕ |
| 手軽さ | ◎（URLのみ） | ○ | ○ | △ |
| SNS適性 | ◎ | △ | ✕ | ✕ |
| コスト | 無料 | 無料〜課金 | 無料 | 無料〜課金 |

**他の選択肢の限界：**
- LINE占い・cocoloni：課金前提、キャラが薄い、友達感がない
- Yahoo!占い：情報量が多すぎ、コンテンツ消費型でつながりがない
- ChatGPT直接：専用UIなし、キャラなし、占い特化でない

## 4. 買った／体験した後の未来（After）

**短期（1回の占い後）：**
- 今日の小さな一歩が決まる感覚（「これやってみよう」）
- 気持ちが整理されてスッキリした
- 友達に送れるネタができた

**中長期（継続利用後）：**
- 朝の習慣になり、一日の気持ちの立て方が変わる
- タロットや星座への興味が深まり、自己理解のツールになる
- こんにゃん堂 = 「安心できる場所」として定着する

**感情的な変化：**
- 安心（「一人じゃない感じがする」）
- 軽さ（「大げさにせず、ちょっと相談できた」）
- 楽しさ（「ツキがかわいい」「ノアの言い方が好き」）

## 5. 刷り込みキーワード（Imprinting Words）

> 全コンテンツで繰り返し使う言葉。
> ブランド定着とトーン統一の核になる。

**主要ワード：**

| ワード | 品詞 | 使用シーン |
|-------|------|----------|
| こんにゃん出ましたけど〜 | フレーズ | 占い結果の冒頭のみ。乱用禁止 |
| 友達感覚 | 名詞 | コンセプト説明・ブランド紹介時 |
| 直感 | 名詞 | 背中を押す場面、カード解釈 |
| 流れ | 名詞 | 運勢説明（「いい流れ」「流れが変わる」） |
| ヒント | 名詞 | 占い結果の位置づけ説明（断定しない） |
| 整理 | 動詞 | 心の整理・気持ちの整理 |
| 気になってた | 動詞 | 行動を促す場面 |
| ツキが | フレーズ | ツキのエピソードを始める定番パターン |
| 24時間 | 副詞 | AIフレンドとしての強みを伝える場面 |

**組み合わせパターン例：**
- 「直感で引いてみて」「今日の流れをヒントに」「ツキが選んだカードがこれ」

**使用上の注意：**
- 「こんにゃん出ましたけど〜」はサイト内占い結果のみ。SNS雑談・note記事では使わない
- 「素敵」「輝く」「キラキラ」「幸せ」は汎用ポジティブとして禁止ワード（→「整理された」「スッキリした」「楽になった」で代替）
- 不安を煽る表現・断定的未来予測は全媒体で禁止
